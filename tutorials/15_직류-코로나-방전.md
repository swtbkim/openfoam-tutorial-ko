## 1. 배경

### 1.1 직류 코로나 방전

방전(discharge)이란, 절연체에 임계치 이상의 전기장이 가해져서 매질의 절연성이 파괴되고 이온화되는 전기적 현상이다. 코로나 방전(corona discharge)란, 뾰족한 전극 주변에 불균일한 전기장이 형성되고, 이 전기장의 세기가 국소적으로 임계치 이상으로 강해져서 방전극 주변의 국소 영역에서 이온화가 발생하는 방전 현상이다. 전극 형상의 곡률이 커서 인가 전압이 낮더라도 전기장의 세기는 국소적으로 임계치를 초과하는 경우에 코로나 방전이 발생하며, 일반적으로 와이어, 핀, 바늘과 같이 곡률 반경이 작은 전극이 방전극으로 사용된다. 방전극 주변의 국소 영역에서만 이온화가 일어나기 때문에 매질 전체의 절연성이 파괴되는 아크(arc) 방전과는 구분되며, 전기집진기, 정전 도장, 이온풍 발생기, 오존 발생기 등 다양한 산업 장비에 활용된다.

코로나 방전은 전자 충돌이나 이온화 과정과 같은 화학 반응, 전하의 전기적 이동, 중성 분자의 유체역학적 수송 등 여러 물리 현상이 복잡하게 얽혀 있는 다물리 현상이다. 전자의 거동은 볼츠만 방정식을 이용해 기술하며, 볼츠만 방정식의 해를 이용하여 전자의 에너지 분포 함수, 전자 충돌 반응의 반응 상수 등을 계산할 수 있다. 그러나 볼츠만 방정식을 이용한 전자 거동의 해석은 길이 스케일과 시간 스케일이 매우 작아, 거시적인 유체 현상과 연동하여 해석하기 매우 어렵다.

단극성 직류 코로나 방전의 경우, 볼츠만 방정식을 이용한 전자 관련 해석 없이 매우 단순화된 모델을 이용해 방전 현상을 유체역학적으로 모사할 수 있다. 단순화된 모델에서는 방전의 결과로 생성된 전하를 단일 이온으로 모사하고, 전자의 거동이나 이온화 반응 등 미시적인 화학 반응 과정은 직접 계산하지 않는다. 대신, 방전에 의한 전하 생성의 효과는 방전극 표면에서의 전하 밀도 경계 조건에 암시적으로 반영하며, 전기장에 의한 이온의 이동(drift)과 그에 따른 공간 전하 분포는 가우스 법칙과 전하 수송 방정식을 이용해 거시적으로 해석한다. 이러한 단순화를 통해 계산 비용을 크게 절감할 수 있으며, 산업용 전기집진기와 같은 대형 스케일에 대해서도 해석이 가능하다.

### 1.2 지배 방정식

코로나 방전 해석을 위한 단순화된 유체 모델은 아래와 같이 2개의 변수(전압 $\phi$, 전하 밀도 $\rho$)에 대한 2개의 수송 방정식으로 구성된다.

$$\nabla ^2 \phi = - \frac{\rho}{\varepsilon_0}$$

$$\frac{\partial \rho}{\partial t} + \nabla \cdot \vec{j} = \frac{\partial \rho}{\partial t} + \nabla \cdot \left( \rho k \vec{E} \right) = 0$$

여기서 $\phi$는 전압($\mathrm{V}$), $\vec{E}$ (=-$\nabla \phi$)는 전기장($\mathrm{V/m}$), $\varepsilon_0$은 매질의 유전율($\mathrm{F/m}$), $\rho$는 코로나 방전으로 생성된 단극성 이온의 공간 전하 밀도($\mathrm{C/m^3}$), $k$는 이온의 전기적 이동도($\mathrm{m^2/(V \cdot s)}$), $\vec{j}$는 전류 밀도($\mathrm{A/m^2}$)이다. 가우스 법칙에 의해 전하가 분포하는 공간의 전기장 분포가 계산되고, 이 전기장을 따라 이동하는 전하의 분포가 전하 수송 방정식에 의해 계산된다.

### 1.3 경계 조건

전압의 경계 조건을 부여하는 방법은 명확하다. 방전극에는 인가하고자 하는 전압의 값을 입력하며, 접지극에는 0 V를 입력한다. 따라서 모두 `fixedValue` 형식의 경계 조건을 사용하며, 일반적으로 경계면 전체에 걸쳐서 균일하게(`uniform`) 입력한다.

반면에 전하 밀도의 경계 조건을 결정하는 것은 쉽지 않다. 단순화된 유체 모델을 이용한 코로나 방전 해석 시, 전자의 거동이나 이온화 반응 등 여러 화학종 간 상호작용의 효과는 직접적으로 계산하지 않으며, 그 종합적인 영향이 방전극에서의 전하 밀도의 경계 조건에 암시적(implicit)인 형태로 나타난다. 따라서 전하 밀도의 경계 조건이 얼마나 적절히 선정되었는가가 단순화된 유체 모델을 이용한 코로나 방전 해석의 신뢰성을 좌우하며, 전하 밀도의 경계 조건을 선정하는 행위 자체가 코로나 방전의 특성을 기술하는 일에 해당한다.

실험적으로 측정된 전류-전압 곡선이 있는 경우, 각 인가 전압 마다 반복 계산을 통해 측정된 코로나 전류만큼의 전류가 발생하도록 방전극에서의 전하 밀도의 경계 조건을 업데이트하여 사용할 수 있다. 이 경우, 코로나 방전의 특성은 실험적으로 측정한 결과를 가져오고, 시뮬레이션은 단지 그 내부의 수송 현상이 어떻게 이루어지고 있는가를 이해하는 목적으로만 사용된다. 측정값이 없는 경우에는 방전에 의한 전하 밀도의 경계 조건에 대한 모델링이 필요하다. 일반적으로는 Kaptzov 가설을 이용해 방전극에서의 전하 밀도의 경계 조건을 예측한다.

#### 1.3.1 Kaptzov 가설

상온 상압의 공기는 절연 물질로, 방전극에 약간의 전압이 인가되어도 공기 중으로 전류가 흐르지 않는다. 그러나 인가 전압 $\phi_{applied}$이 코로나 방전 개시 전압 $\phi_{on}$에 도달하여 방전극 주변의 전기장 세기 $E$가 코로나 방전 개시 전기장 세기 $E_{on}$에 도달하면, 공기가 국소적으로 이온화되어 코로나 방전이 개시되고 방전극과 접지극 사이에 공기를 따라 코로나 전류가 흐른다. **Kaptzov 가설**이란, 일단 방전이 개시된 이후에는 인가 전압이 더 증가($\phi_{applied} > \phi_{on}$)하더라도 공간 전하의 차폐 효과에 의해 방전극 표면의 전기장 세기가 $E_{on}$으로 일정하게 유지된다는 가정이다. 따라서 Kaptzov 가설로 코로나 방전을 해석할 때는 방전극 표면의 평균 전기장 세기가 $E_{on}$이 될 때까지 방전극 표면의 전하 밀도 경계값을 반복적으로 업데이트하며, $E_{on}$은 일반적으로 Peek 방정식으로 계산한다.

![Kaptzov 가설 : 인가 전압에 따른 방전극 표면에서의 전기장 세기 변화. 공간 전하가 없는 라플라스 필드는 인가 전압에 비례하여 선형적으로 증가하지만, 코로나 방전이 개시된 이후의 푸아송 필드는 공간 전하의 차폐 효과에 의해 $E_{on}$으로 고정된다.](../figures/15_kaptzov.png)

#### 1.3.2 Peek 방정식

Peek 방정식은 방전극의 형태, 크기, 방전 조건에 따라 방전이 개시되는 순간 방전극 표면에서의 전기장 세기를 구하는 계산식이며, 아래와 같이 주어진다.

$$E_{on} = E_b f \left( 1+ \frac{0.0308}{c_0 \sqrt{r}} \right)$$

여기서 $E_{on}$은 코로나 방전 개시 전기장 세기($\mathrm{V/m}$), $E_b$는 매질의 절연 파괴(breakdown) 전기장 세기($\mathrm{V/m}$)이다. 상온·상압에서 공기의 절연 파괴 전기장 세기는 약 3.0e+6 V/m로 알려져 있는데, Peek 방정식을 이용하여 코로나 방전 개시 전기장을 계산하는 경우에는 일반적으로 $E_b$ = 3.1e+6 $\mathrm{V/m}$을 사용한다. $f$는 방전극의 표면 상태에 관한 계수로, 기계적으로 연마된 깨끗한 금속 표면은 $f$ = 1.0이고, 일반적인 연구실 실험 조건에서 반복적으로 사용된 방전극에 대해서는 약 0.5--0.7로 알려져있다. 즉, 표면이 거칠수록 마이크로 방전 등에 의해 더 약한 전기장에서도 코로나 방전이 개시된다. $c_0$는 방전극 형상 변수로, 와이어 등 원기둥 형태의 방전극은 $c_0$ = 1.0, 핀이나 바늘 형태의 방전극은 $c_0$ = 0.5이다. $r$은 방전극의 곡률 반경($\mathrm{m}$)으로, 와이어 방전극에서는 와이어의 반경을 사용하고, 핀이나 바늘 형태의 방전극에서는 방전극의 첨단의 곡률 반경을 사용한다.

## 3. 해석

### 3.1 전처리

#### 3.1.1 예제 복사

아래의 예제를 다운로드하여 작업 디렉토리로 복사한다.

```bash
$FOAM_TUTORIALS/electromagnetics/electrostaticFoam/chargedWire
```

#### 3.1.2 0/ 디렉토리

전압과 전하 밀도의 경계조건은 각각 phi, rho 파일에서 정의한다. 전압의 단위는 V = J/C = $\mathrm{kg\,m^2\,s^{-3}\,A^{-1}}$이고, 전하 밀도의 단위는 $\mathrm{C/m^3}$ = $\mathrm{A\,s\,m^{-3}}$이다. 방전극 표면 경계면 `hole`에 45,000 V를 인가하고, 접지극 표면 경계면 `up`에 0 V를 인가한다.

```cpp
//**************************************//
dimensions  [1 2 -3 0 0 -1 0];

internalField   uniform 0;

boundaryField
{
    // ......
    down
    {
        type            symmetryPlane;
    }
    up
    {
        type            fixedValue;
        value           uniform 0;
    }
    hole
    {
        type            fixedValue;
        value           uniform 45000;
    }
    // ......
}
//**************************************//
```

방전극 표면 경계면 `hole`에는 경계값 3.57e-05 $\mathrm{C/m^3}$를 입력하고, 접지극 표면 경계면 `up`에는 `zeroGradient` 조건을 부여한다. 방전극 표면 경계에서의 전하 밀도의 경계값 3.57e-05 $\mathrm{C/m^3}$이 적절한가에 대해서는 후에 논의한다.

```cpp
//**************************************//
dimensions  [0 -3 1 0 0 1 0];

internalField   uniform 0;

boundaryField
{
    // ......
    down
    {
        type            symmetryPlane;
    }

    up
    {
        type            zeroGradient;
    }

    hole
    {
        type            fixedValue;
        value           uniform 3.57e-05;
    }
    // ......
}
//**************************************//
```

#### 3.1.3 constant/ 디렉토리

physicalProperties 파일에서 매질의 유전율 $\varepsilon_0$과 이온의 전기적 이동도 $k$를 입력한다. 예제의 파일에는 공기의 유전율(8.85419e-12 $\mathrm{F/m}$)과 상온·상압 공기 방전 시 발생하는 이온의 평균적인 전기적 이동도 값(1.6e-4 $\mathrm{m^2/(V \cdot s)}$)이 저장되어 있다. 이 예제에서는 값을 수정하지 않고 사용한다.

```cpp
//**************************************//
epsilon0        8.85419e-12;

k               0.00016;
//**************************************//
```

#### 3.1.4 system/ 디렉토리

blockMeshDict 파일에서 해석 형상과 격자계를 정의한다. 이 예제는 와이어-평판 형식의 전기집진기 내부의 코로나 방전 현상을 해석한다. 와이어 방전극의 반경은 $r_w$ = 1 mm이다. 방전극-접지극 간 거리는 114 mm이고, 방전극 간 거리는 152 mm이다. z 방향 깊이는 0.5 m [0, 0.5]이다. 대칭 조건을 이용하여 반복 단위의 1/4 형상만 해석한다. 해석 영역을 여러 개의 사면체로 분할하여 hex block을 구성하고 정렬 격자를 생성한다. 와이어 표면이나 와이어 주변의 고해상도 격자 영역 등을 구성하는 곡선은 arc로 정의한다.

```cpp
//**************************************//
vertices
(
    (0.001 0 0)
    (0.038 0 0)
    // ......
);

blocks
(
    hex (5 4 9 10 16 15 20 21) (60 30 1) simpleGrading (5 1 1)
    hex (0 1 4 5 11 12 15 16) (60 30 1) simpleGrading (5 1 1)
    // ......
);

edges
(
    arc 0 5 origin (0 0 0)
    arc 5 10 origin (0 0 0)
    // ......
);

boundary
(
    // ......
    up
    {
        type patch;
        faces
        (
            (7 8 19 18)
            (6 7 18 17)
        );
    }
    hole
    {
        type patch;
        faces
        (
            (10 5 16 21)
            (5 0 11 16)
        );
    }
    // ......
);
//**************************************//
```

![계산 영역 : z = 0 평면 상 절점의 위치와 hex block](../figures/15_domain.jpg)

controlDict 파일에서 해석 조건과 샘플링 조건을 정의한다. $\Delta t$ = 5e-5초로 총 0.02초 간 해석하며, 0.005초마다 해석 결과를 저장한다. 방전극 표면에서의 평균 전기장 세기를 샘플링하기 위해 `functions`를 정의한다. `Emag` = $|\vec{E}| = |- \nabla \phi|$으로 전기장 세기를 계산하고, hole 표면에 대해 `Emag`의 `areaAverage`를 계산한다.

```cpp
//**************************************//
application     electrostaticFoam;

// ......

functions
{
    gradPhi
    {
        type            grad;
        libs            ("libfieldFunctionObjects.so");

        field           phi;
        result          gradPhi;

        writeControl    timeStep;
        writeInterval   100;
    }

    Emag
    {
        type            mag;
        libs            ("libfieldFunctionObjects.so");

        field           gradPhi;
        result          Emag;

        writeControl    timeStep;
        writeInterval   100;
    }

    Emag_hole_avg
    {
        type            surfaceFieldValue;
        libs            ("libfieldFunctionObjects.so");

        regionType      patch;
        name            hole;

        operation       areaAverage;
        fields          (Emag);

        writeControl    timeStep;
        writeInterval   100;

        log             yes;
        writeFields     no;
    }
}
//**************************************//
```

### 3.2 격자 생성 및 해석 실행

아래의 명령어로 격자를 생성하고 솔버를 실행한다.

```bash
blockMesh
electrostaticFoam
```

controlDict 파일에서 `fuctions`로 정의한 샘플링 결과가 터미널로 출력된다.

```bash
    functionObjects::grad gradPhi writing field: gradPhi
    functionObjects::mag Emag writing field: Emag
surfaceFieldValue Emag_hole_avg write:
    areaAverage(hole) of Emag = 5.48345e+06
```

즉, 방전극 표면에서 전하 밀도의 경계값을 3.57e-5 $\mathrm{C/m^3}$으로 입력한 해석 결과, 방전극 표면에서의 평균 전기장의 세기는 5.48345e+6 V/m로 계산되었다. Peek 방정식에 의하면 $r_w$ = 0.001 m, $f$ = 1.0에 대하여 코로나 방전 개시 전기장 $E_{on}$는 6.1193e+6 V/m이다. Kaptzov 가설에 의하면 코로나 방전이 개시된 이후에는 방전극 표면에서의 전기장의 세기는 코로나 방전 개시 전기장 세기 $E_{on}$으로 일정하게 유지된다. 따라서, 방전극 표면에 전하 밀도의 경계값으로 과도하게 큰 값을 입력하여 방전극 표면에서의 전기장 세기가 과도하게 작게 계산되었다고 판단할 수 있다. 방전극 표면 전하 밀도의 경계값을 보다 작게 입력하여 방전극 표면에서의 평균 전기장 세기가 $E_{on}$ (= 6.1193e+6 V/m)으로 계산되도록 조정한다.
